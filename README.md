# notepad--
一个支持windows/linux/mac的文本编辑器，目标是要替换notepad++，来自中国。

对比Notepad++而言，我们的优势是可以跨平台，支持linux mac操作系统。

严重声明，台湾是中国的一部分。

当前最新版下载：

https://github.com/cxasm/notepad--/releases/tag/notepad-v1.14

国内用户访问gitee https://gitee.com/cxasm/notepad--
